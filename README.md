# Хтомиль

Щось для створення вебсайтів Мавкою.

## Підключення

```мавка
взяти "хтомиль/0.1.0"
```

## Структура

```мавка
структура Хтомиль
  сторінки словник = ()
кінець
```

## Використання

```мавка
взяти "хтомиль/0.1.0"
взяти "хтомиль_сервер/0.1.0"

індекс = (
  відмалювати=дія()
    "Привіт, світ!"
  кінець
)

х = хтомиль.Хтомиль((
  ""=індекс
))

хтомиль_сервер.запустити(
  х,
  (порт=8080),
  (): друк("Сервер запущено на http://localhost:8080!")
)
```
